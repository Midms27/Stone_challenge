Como rodar

1- É necessário ter instalado node em sua máquina, link para download abaixo:
https://nodejs.org/en/

2- Abra o terminal e certifique-se de estar na pasta do programa, no caso a pasta app, o final do path deve se assemelhar a isso:
stone_challenge\app

3- execute o comando node main.js

Para teste

Os arquivos utilizados para teste estão dentro da pasta data, são dois arquivos em formato JSON, um para os produtos e outro para emails.
Para testar com os produtos, use objetos com as chaves product, quantity e price(sempre os adicione dentro do array), ou mude os valores dos obejtos no array.
Para testar os emais, simplesmente adicione emails no array ja existente.

Importante 

Para fins do desafio todos os valores dos produtos são multiplicados por 100 para evitar erros
